Jewellery brand application with loyalty programme

By Ho Kei Leung

4 weeks | Solo project


Introduction:

Bling: is an Jewellery brand application with loyalty programme for customer to shop and review their account activities

The Problem:

With consumers becoming increasingly reliant on their mobile devices and shopping apps. Bling wants an application that their user can shop and implement a loyalty programme for the brand to enhance shopping experience.

Goal:

The goal is to create an application for user to be able to shop, review orders and their loyalty programme status all-in-one using the application.

Design Process


User Research Insight


A primary user group identified through research were young adults who prefer to shop and browse using mobile.

This user group confirmed initial assumptions about the jewellery brand application customers, but research also revealed that time was not the only factor limiting users from browsing using mobile. Other user problems included obligations, interests, or challenges that make it difficult to shop and track their account activity using desk tops.
